Unit 8 Activity: Data and Persistence

This unit covers JSON, cookies, Web Storage, and saving data to a file from the browser.

Exercises

  • JSON parsing and stringifying
  • Cookies and personalised messages
  • Local Storage and Session Storage
  • Persist data to the filesystem

What This Demonstrates

  • Representing structured data with JSON.
  • Saving simple client-side values using cookies.
  • Persisting key:value data with local and session storage.
  • Exporting JSON to a file and reading it back in the browser.
