Unit 9 Activity: Sound and Vision

This unit covers HTML audio, sound synthesis with the Web Audio API, and drawing with the HTML canvas.

Exercises

  • Audio element and custom controls
  • AudioContext synthesis and simple piano
  • Canvas 2D drawing examples
  • Combined sound and vision demo

What This Demonstrates

  • Playing sound using the HTML audio element.
  • Generating tones from scratch using AudioContext.
  • Drawing shapes and scenes with the 2D graphics context.
  • Combining sound and visuals in one interactive page.
